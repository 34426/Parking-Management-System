## 一、介绍

运行环境:idea或eclipse vscode 数据库:mysql

开发语言：java

前端技术：vue、element-ui

后端技术：springboot、mybatisplus

系统有两个角色：管理员和用户

一、 管理员

用户信息管理

车位信息管理

车位费用设置

停泊车辆管理

车辆进出管理

登录日志查询

二、车主

个人中心

预定停车位

缴费信息（停车缴费记录、待缴费、账户余额充值）
